PS - I use email addresses as my primary user ID.  When the user logs
with a *new* OpenID, I bind it to an email address.  They can later
migrate to a different OpenID, no problem.

Google also uses email addresses.  So it would be easy for me to offer
a Google login box, as an option.  The user could later switch to
OpenID, seamlessly.

But I expected Google would have a standard OpenID scheme for their
users.  Other sites have them.  So, for instance, if you have a
LiveJournal account, you get this OpenID, which you can use on other
Web sites:

  username.livejournal.com

  (it then authenticates thru LiveJournal)

I can't find the equivalent for Google.
