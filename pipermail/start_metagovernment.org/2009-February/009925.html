I hear your frustration, but I don't agree that nothing is being done.
We have a few developers who are working steadily towards the first
release - it's not a trivial endeavour. I know because I've been
independently coding my own metascore-ish type system for the last two
years.

I've found that it's not like designing a word-processor, it's like
designing the first ever word processor - there can be a bewildering
number of design choices, without precedent. Because of this, simply
throwing more developers at a v1.0 problem doesn't always help.
