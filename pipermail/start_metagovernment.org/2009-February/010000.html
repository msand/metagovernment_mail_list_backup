Everyone on this list is requested to please visit the Metascore  
prototype:
http://prototype.metascore.org/
and play with it in order to give feedback to the developers. You can  
leave feedback on Metascore or on this list.

If you get lost, Manuel has made a video demonstration of the workflow  
here:
http://vimeo.com/3143664

Yes, the interface is sorely lacking (and the scoring and tagging  
systems are not available yet). If anyone is able to contribute time  
to working on the interface, that would be greatly appreciated.  
Alternately, if you know of someone or of a place where you might be  
able to ask around for someone, please make that contact and try to  
recruit an interface designer to the project.

Thanks.
