We had a meeting today with a new crew of developers and the decision was
made to move the code from Google's App Engine to servers at Oregon State
University's Open Source Lab.  This removes a few technical barriers and a
significant political obstacle in recruiting new developers (nobody seems to
like Google).  Open ID is a desirable login mechanism, but is a ways down
our list of development priorities (a copy of which I'll forward soon).
