I agree that we may be ready to move on to considering further 
proposals. The way to do that (so long as we are using emails) would be 
to put each new proposal into an email, and use the email to start a new 
thread.

On the current thread: The question that we have been considering has 
been whether to use voting *OR* discourse as the means through which to 
reach public decisions. Have we settled that matter? (My proposal has 
been that we reach public decisions through discourse, without voting.)

(This is a big deal. Pretty much all of the decision system projects on 
the Metagovernment list use voting.)

If anyone following this thread still thinks that we should use voting 
to make public decisions, now would be a good time to speak up and 
explain why. (In a vote-driven decision system, reasons remain private 
and have their effect through the cast vote; in a discourse decision 
system, reasons have their effect through being communicated to others.)
