I will be brainstorming (and reporting to the list) traits that
classify kinds of decisions in an abstract level (common to any
organization). It would be nice to find the smallest set of
characteristics that no different kinds share exactly the same values
for these characteristics.

By time:
Decisions concerning past, present and future; respectively
examination, operation and planification.

By input:
As Pietro said, yes-no decisions and multiple choice decisions.
