I couldn't follow up on this at first because the extension download
site was broken.  I finally got back to it yesterday.  I installed
both extensions on the pollwiki and dropped all other spam defences,
including the captcha.  I was immediately hit by one spammer.  Today I
added a local blacklist and (for good measure) Wikipedia's list:

  require_once( "$IP/extensions/SpamBlacklist/SpamBlacklist.php" );
  $wgSpamBlacklistFiles = array
  (
    # http://zelea.com/w/MediaWiki:Spam-blacklist,             # local (implicit)
     'http://en.wikipedia.org/wiki/MediaWiki:Spam-blacklist',  # well maintained list
     'http://meta.wikimedia.org/wiki/Spam_blacklist',          # default list
     'http://metagovernment.org/wiki/MediaWiki:Spam-blacklist' # neighbouring wiki
  );

I also added a local list to metagov.  Ed, you might want something
like this in your own LocalSettings, just below the require_once:

  $wgSpamBlacklistFiles = array
  (
    # http://metagovernment.org/wiki/MediaWiki:Spam-blacklist, # local (implicit)
     'http://en.wikipedia.org/wiki/MediaWiki:Spam-blacklist',  # well maintained list
     'http://meta.wikimedia.org/wiki/Spam_blacklist',          # default list
     'http://zelea.com/w/MediaWiki:Spam-blacklist'             # neighbouring wiki
  );

Still testing with a wide open pollwiki ...
