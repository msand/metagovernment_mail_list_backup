Its not about a certain date, its continuous, once a majority adopted, it
can be counted as "in effect", once a proposal does not any longer share a
majority, it goes "out of effect".
Github as sucht has nothing to do with voting, just the mechanisms it
offers can be used to play around with it and you can create a github
account with others and use that kind of methodology and eventually an
"APP" using the github API can be created to make that sort of usage easier.

> > 1. Every times  "PULL REQUEST" is made, a PROPOSAL is made
