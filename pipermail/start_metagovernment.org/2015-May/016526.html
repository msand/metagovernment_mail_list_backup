I find github very interesting, and I dream the moment when we can
integrate vilfredo as part of github, so that for specific questions a
vilfredo question is opened. What I wonder is: how democratic is a system
where the only person that can participate in the discussion really (that
is with direct independent power) are the programmers who can fork a
program?

I don't have an answer, but I wonder...
