Code will be law. A repository is/can be a proposal.
Someone creates a repository for anything and people discuss and
participate and either fork it or create pull requests.

For anything that has to be discussed in a serious way, you need
professional tools like version control.
Currently only programmers use it, but the circles of people using github
grow bigger and it becomes more user friendly.
There are all kinds of tools around github or git systems that make it
easier.
