Never read one of those, but I figured if I'm going to build a system
to replace it I'd better get a better idea how it's been done in the
past.  The github distribution I didn't find very convenient, but the
same data is available directly on the whitehouse.gov web site
(http://www.whitehouse.gov/omb/budget/Overview).  What's the point of
distributing it on github if you're not going to allow people to tweak
it and then merge changes back in?

The budget itself isn't all that informative because it leaves out any
discussion of the largest expenses (the massively bloated defense
budget, that weird postal-service charge that isn't described in the
text at all, etc).  Not to mention that there is never any indication
that our input has been solicited to change any of the proposed
values.  Is the budget part of our democracy or not?
