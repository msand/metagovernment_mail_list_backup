I did tons of discussions in my life.
1. The content or results of those discussions are lost if not written down.
2a. Progress is slow, because without writing it down and having some
discipline - you need to bring the same arguments over and over until
repetition has helped the listeners to remember
2b. Only then can you continue and build upon which has not only been
agreed on by people but also remembered that is has been agreed upon.
... thats slow as hell!

YES - beaurocracy in the sense of lots of procedures and paper is HORRIBLE!
Thats why we have computers to automate that. It's now all about automated
beaurocracy.

The intellectual world will be shaped by software and those who are able to
write it.
If you want some influence - learn to code - thats what i would suggest you
and everyone else here.

If you never did, i suggest you:
FIRST: www.codecombat.com (select javascript)
SECOND: www.codecademy.com (go through some HTML, CSS and JAVASCRIPT
lessons)
THIRD: https://pcottle.github.io/learnGitBranching/ (learn basic concepts
behind GIT)
and if you want to now really start...
FOURTH: www.nodeschool.io (learn the basic concepts behind NODEJS)

Then join which ever projects you like, there are zillions on github and
its happening now.
In addition tell the people you know to start learning today (see above
links)
