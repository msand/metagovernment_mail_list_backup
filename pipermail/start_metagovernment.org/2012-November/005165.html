WHAT?

**Frauke Mattfeldt** presents the concept and idea of **probble.de**:

probble! is a problem solving platform for the collaborative solving of 
problems.

For a short overview here are the basic functionalities of the platform 
listed:
- Description of a problem
- Adding documents, images and YouTube-Videos to the description
- History of the description (like Wikipedia)
- structure of a problem (parent and child problem)
- Categorize and localize the problem
- Google map function
- Adding ideas for the solving of problems
- Adding documents, images and YouTube-Videos to the idea
- Supporting a problem and an idea
- Connecting problems with each other
- Adding events to the problem (a kind of calendar function)
- Searching for problems based on the following properties: location 
(country, state), category, date, supports

See http://probble.de/ (unfortunately in german only) for more information.
