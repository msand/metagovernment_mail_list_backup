If you use a decent email client, the mailinglist is already like a forum.
I'm using google mail and it groups conversations relatively reliable.
The german pirate party uses "sync forum", a tool, that synchronizes
mailinglists with a forum, so that people can use either way of
communicating.

Neither email nor forum do solve the problem of presenting "current state
of consensus" (aggrement/disagreement) in a discussion or let's just say
"summary", so that focus is kept and discussion does not drift in the wrong
direction.
