Thanks for forwarding this, Pietro! To those on this list, I should
also mention that while we're currently talking about tracking tools
(i.e. gov monitoring), the plan is to add on tools (e.g. drafting and
voting) for a r/w approach, leading into Adhocracy2 (Adhocracy
1.1/liqd.net [1] does most of this already but it has come of some age
and is in need of a cleaner way to handle branched and versioned draft
text.)

We would be happy about any collaborators from MetaGov!
