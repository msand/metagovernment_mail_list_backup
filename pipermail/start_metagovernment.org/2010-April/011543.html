Open source is of course the condition to a trusty and transparent eGov, 
but it is not sufficient. A running program like a website can run with 
a supposed open sources, but in fact be modified by a malicious admin. 
We need also a kind of trusting sources process that continualy attests 
that the running program is identical to the trusted sources.
