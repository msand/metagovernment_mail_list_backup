On the one side we clearly want to have everything firmly established on
open source soil. On the other companies can sometimes produce rapid result
that in the short run might win the competition. Until the problems with
having companies being responsible for public roles come up. I think here
about the voting machines, for example.

Maybe we could do something as a metagovernment group. And you Richard could
also help us from the GNU. And maybe we can get even more people to sign
in.

We could write an email of recommendations taking a balanced and open
position. Something that does not rule out the private sector but requires
private companies to use only open source tools.

Eventually we are going to have open source tools in those places simply
because the first nations will use the closed versions and then start
tripping on their own feet. Like SO using the fact that Obama is so in favor
of eGov, we should suggest that he should not do the same error that have
been done with voting machines. And should start directly with open tools.

Any thoughts?
