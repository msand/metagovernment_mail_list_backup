Hello Paappo and everyone:

I work in an open source decision-making voting system called
agoravoting [1]. Currently it's a voting system but we plan to extend
it to allow a wider spectrum of the decision-making process. It has
been already used by spanish congressmen [2] and will be used again
soon, and we're now adding secure and verifiable elections support.

If you want to help with translations we'd very much appreciate so [3] =)
