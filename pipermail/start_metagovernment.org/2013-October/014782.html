Actually,what you are trying to do is EDUCATING people :-) ...to see a
solution to a problem.
Could you write down the problem you are trying to solve?
...or at least ONE ASPECT of the problem, to which ONE ASPECT of your
solution might actually fit? :-)

btw... in order for something to be a solution, it has to respect
"usability issues" too.
1. If something is easy to use, it educates people about a solution
2. If something is easy to use, it changes the path of least resistance, so
people will naturally follow it :-)

A theoretical solution might be so hard to use, that it doesnt work in
practice.
"Nuclear Fusion" in theory is awesome... but in practice, all the
"solutions" in existance are so bad, that they actually cannot provide the
benefits which the theory promises... (at least not yet)
