> All citizens should vote.
> Uninformed citizens should not vote.
> 
> I ask us to treat both propositions as true and solve the conflict by semiforcing thoughtfulness.
> 
I would rather lean heavily toward the first one. In fact, I take it further: non-citizens, criminals, children... *everyone* should be allowed to vote if they feel that have an interest in that particular decision.

There is no need for controlling for the ignorant, because we're not talking about majority rule systems. In a consensus-based system, the best ideas are the only ones that have any chance of winning.
