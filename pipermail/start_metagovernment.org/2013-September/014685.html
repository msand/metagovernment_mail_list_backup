Is it possible, that some "messages" of a discussion end up in different
"topical public mirrors"?

Those "gate keepers" that sit between the actual discussion and are able to
"ack" or "nak",
can decide which image is created in a "public mirror", which again will be
reflected in further discussions
and therefor influence how discussions might evolve.
So the mechanism is a very important one.



>

existance,
how many "public mirrors" can exist for one community,
whether a message in a discussion can be part of multiple "public mirrors"
whether it can even be part of different "topical public mirrors"
and what all the other mechanisms might be...

...as soon as i grasp all the mechanisms involved, then i can try to do
some kind of
"monte carlo simulation" in my head, with all the rules involved, to find
out
how such a "system" might behave :-)
