I think there should be a 'group' option -- where groups can organize 
themselves and optionally choose leaders whose votes as 'group votes' on 
particular issues -- given a scoring similar to that of individuals, 
weighted according to the size and popularity of the group. Also, groups 
should be able to govern themselves using the same software as the 
government. They can develop legislative proposals or position 
statements within the group, then open the statements for public 
comment. Also they should be able to limit which users can become 
members, but all people can 'approve' or 'disapprove' of the groups actions.

This could be something analogous to facebook groups -- perhaps we can 
interface with facebook to accomodate this??
