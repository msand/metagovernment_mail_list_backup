If we let all data in an installation be open, and offer apis, so that
other installations can replicate the other installation, anybody
could download the version of metascore that the server they
wish to audit, claims to use. If the software on their system
behaves as that on the system being audited, one can be
fairly certain, that the system hasn't been tampered with.

These apis could also be used, to implement the second two
features you mentioned.
