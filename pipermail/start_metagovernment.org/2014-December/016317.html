i think it boils down to DOING, more than it boils down to TALKING :-)
In the end, i imagine, that people who are inspired start their own
projects and push no matter if someone follows or not. they are willing to
do it all alone and some might join forces.
Maybe some are developers, maybe some are good in raising some funding or
creating some kind of marketing buzz.
After everyone has her or his project, there can be discussions of how to
"join forces", by connectin projects via APIs :-)

There is currently one project that pushes really hard
https://github.com/fairdemocracy
There might be more projects and if they were ALL on GITHUB then it would
be even more easy to collaborate, because there is this AWESOME CHAT
SERVICE called http://gitter.im, which allows to make chats for each github
project AND if you join multiple projects, you can see them all in your
sidebar AND in each of those projects, you can configure in settings, that
notifications about any kind of PROGRESS in any of those projects is shown
in the other sidebar, thus:

vilfredo updates will show in the "activity stream" of each of the
collaborating projects as all the other acitivites will be shown too.

*@all*
if everyone would try to make a GITHUB ACCOUNT and at least join one
project as a spectator, that would move us forward a lot.

*@all non programmers*
You can create a "Github Organisation" and stat a "Github Project" to open
a "Gitter Chat" and maybe even a "waffle.io" project board, which offers
cool free of charge project management and then just brainstorm and write
down the concept, so that sooner or later, a programmer might join and help
building it :-)
just give it a try.
