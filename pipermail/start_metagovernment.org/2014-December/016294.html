Meh, in the current zeitgeist political "consent" basically means
"resignation" (i.e., apathy, disenchantment, disillusionment,
alienation, etc.).  It's not that people fundamentally disagree with
the philosophy that "every vote counts", it's that they instinctively
realize that the return on invest for any action they take is so small
as to ensure that it's not worth the effort.  IMHO the only way to fix
this is to lower the barriers so far that even a trivial amount of
effort really does make a difference (e.g., the Matchism upgrade has a
lot more on proxies and individuals participating in the process of
setting goals for the government than The Plan did.  And this will be
the main emphasis for The System which I hope to be able to resume
working on soon).
