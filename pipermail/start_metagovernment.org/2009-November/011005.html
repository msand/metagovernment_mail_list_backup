Thanks Jacki,
I thought about it, and I reached the conclusion that the way to use Vil in
this context would be to either ask a different question (what should we
write here....?, for example), or run Vilfredo only for one generation.
Think about it, Each generation gives you the pareto front of the answers.
SO you are getting a set of answers that represent the community. That is
what YOU wanted to know: where are you now.
Then the genetic algorithm part of Vilfredo, the fact of feeding the answers
back, and getting people to write more, and so on, is more the part of the
algorithm that is trying to find a workable middle ground. But your question
was not about the middle ground, you were really asking our opinions. Not a
compromised point of view.

And all this suggests that maybe Vilfredo should include the option to let
the questioner close an answer before we reach a consensus, and define
himself as satisfied with the answer he received.

Cheers,
Pietro
