Ah, but that is a level of refinement that I would argue is in a
different category than the valence (+/-) of the value itself.  It's
like balancing a pin on it's tip, from the zero point it is simply
neutral, but nudge it one way or another (regardless of the level of
refinement) and it selects a much vaster category than the (later)
finer levels of intensity *within* a given valence (in this analogy,
either the force of gravity or the length of the pin would have to be
variable, but in either case the *valence* (opposed or supported) is
retained).

Again, I think this is only a problem with entertaining multiple
values within a given valence, not the three-valued logic as given.
(It may be interesting to note that in digital circuits, for example,
having a three-valued system of true, false, and "don't care", many
operations are made much more efficient.  Adding values beyond this
add nothing.)

Each user should be seen as it's own dimension and each user is adding
a vector of force along their own dimension or "angle" from that
delicate, (as yet) undecided zero-point.  The intensity of that force
along any given dimension is a factor of the user's reputation.  The
factor of time, or generations, is like this:  For each given
proposal, it is already a winning proposal, a *working* proposal whose
confidence is set to the propos-ers's reputation, but upon any dissent
it is marked "tentative".  Anyone can dissent at any time like
Vilfredo to start the process of further re-evaluation/synthesis, at
such point the proposal is marked "in contention", but can still be
allowed to be a working proposal.

There is some fascinating number theoretical (and geometrical)
problems in here, but too cumbersome to go into...

The ignore part drops out and disappears into (or as) a separate
dimension.  This is where the recursive, fractal function comes into
play because presumably these votes are occuring within an even larger
context or set of axioms/principles; in other words, ultimately all
the proposals must relate to these foundational principles (either in
support or opposition to them), like the metagovernments [[basic
principles]], for example, or the Universal Declaration of Human
Rights.

In any case, and in this sense, it is important not to let the approve
part dominate the oppose part but keep each as differently valenced
forks in the decision tree, letting each evolve and decend/branch-out
into cogent elucidation of the controversy on both sides *until* one
or the other branch is seen to be in contradiction to a foundational
principles.  If no such contradiction is found, presumably, it is in
support of them.  Obviously if they are in opposition to said
principles, the proposal must be rejected (that branch of exploration
is dropped from the trunk and the points/value invested in it gets
recycled back into the community).  In the event of support, the
proposal is added off the main trunk, with all the vote points and
detail propagated down its "branch" to constitute its public record.

As you can see, I'm taking the voting methodology and making it into
an abstract economy of ideas, a "Tree of Life" with the foundational
principles as the trunk of the tree.

> If you have two questions, and everybody approve the first, and you ignore
> the second, does this mean that the first dominate the second, and the
> second should be lost?

No, both are kept, but the value (or confidence) assigned is the
summation of votes in support.  Consider it like the "thickness" of
the branch of the aforementioned Tree.  It represents the strength of
the proposal in relation to the foundational principles of the trunk
of that community.

> What if instead everybody opposes the first and ignores the second, whould
> we now keep the second?

The first is kept until a contradiction is found further up the tree
that would put it at odds with the foundational principles (only then
could it be "dropped").  Until then, the proposal is simply marked
"controversial".   The second is still kept.

Again, I think the only way to look at this coherently is within the
context of a *larger* set of (more general) axioms in which the
proposals are occurring.  Only within a certain scale is it one
dimensional, but in the larger context it is part of an enfolding of
propositions that is multidimensional.
