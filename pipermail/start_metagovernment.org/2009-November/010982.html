Some answers that I submitted to the "wall of text" question in Vilfredo,
now lost for some reason. Fortunately I did a backup, because I considered
some ideas interesting (the last one).


  An icon or an emphasized title/link for those proposals that can be read
within a minute or less at normal reading speed.


Prizes or reputation points for the succinct writer.


Collaborative resuming: you have a marker and want to read entirely that
long, long, proposal; using it you can point to others what was significant
to you.


Drawing the proposal whenever possible. "A picture tells a thousand words."

I remember that thing one never would use: yes, (Java) Universal Modeling
Language (UML). Maybe we can create something more useful, the Proposal
Modeling Language.


Shorter proposals appear in the first places of a ranking. There are two new
buttons: I understand, I don't understand. Negative understanding points
sink the proposal in that ranking.


Writing an abstract of the proposal.
