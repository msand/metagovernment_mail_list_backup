> Hi

> We have been working on candiwi for quite some time now and currently
> have a very usable version. Manu has set up an instance for
> Metagoverment on his server. I will deliberately keep the explanation
> of how it works short to see how usable it is so please feel free to
> ask questions if something is not clear or if bugs are found :).
>
> Basic Usage:
> 1. To change articles in the main repository create a proposal.
> 2. Edit article(s) in the proposal until happy with the new state.
> 3. Submit proposal for voting. If it is accepted the changes will be
> applied to the main repository.
>
> The main thing to understand is that their is a canonic version which
> can only be changed via proposals.
>
> Important: Their is a "magic" article called "settings" which contains
> the voting rules for the community and it is used by candiwi to
> determine if a proposal will be accepted or not. It can be edited in a
> proposal like everything else.
>
> http://metagov.candiwi.org/
