The Pareto Front is not the minimum set of proposal such that everybody is represented. The fact that everybody is represented is a (lucky) side effect but was never designed to be so. The pareto front is the set of proposal undominated. Now neither 2790 and 2799 dominate 2741. You could probably say that 2790+2799 dominate 2741, but then also 2799+2741 would dominate 2790 and 2790+2741 would dominate 2799. 

So since no single proposal dominate 2741, it is in the Pareto Front.
