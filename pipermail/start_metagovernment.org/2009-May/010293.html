I think my day is about to get busy, so I don't have time to start an  
article on the wiki now, but it seems like the wiki would be the best  
place to put together the groundwork for a global API for lawmaking/ 
policymaking.

Here are some initial ideas for laying out the groundwork for the  
format, if someone would like to take it further... or come up with  
something completely different. I'm no programmer, so don't know if  
this is the way to start.

Act (or Law or Policy or Bill, etc.):
- Title
- Summary
- Sections - which may include but are not limited to:
-- Prohibitions - Things people are prohibited from doing
-- Compulsions - Things people are required to do
-- Rights - Things which people are allowed to do
-- Proactions - Things which the governing body is required to do
-- Resolutions - Things which the community holds to be true
-- Funding
-- Enforcement
- Paragraphs

Tracking Information
- Version
- Timestamp
- Containing community (ie, jurisdiction)
- Author

Comparison Information
- Issues this act pertains to
- Other Acts which this Act:
-- Contradicts
-- Supersedes
-- Synthesizes
-- Obsolitizes

Status
- (not sure how to break this down in a way that works with various  
systems)

Discussion
- (not sure how to break this down logically)
