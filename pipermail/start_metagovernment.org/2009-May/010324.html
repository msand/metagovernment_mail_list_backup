Trying to cover the difference between Pietro and Ed - between
localized polling on local issues, and universal polling on the same
issues.  What if there's no up-front restrictions in the underlying
voting network - anyone can vote on anything - but then we filter the
votes at a higher layer?  Ideally the filters are per user, so it's
like you have magic spectacles to impose your own eligibility criteria
on the voters, without imposing them on everyone else.

If we can code that (eventually!), will it work for both?
