It's easy! :-)

I suggest, that any of you tries to:
1. learn the basics of "markdown"
2. make a pull request (think: "change proposal") to our metagovernment
community repository (think "wiki")
=> It should contain a small tutorial about how to do what i describe below
:-)


*GET STARTED:*


*Click this link and see https://github.com/Metagovernment
<https://github.com/Metagovernment> the metagovernment organization has
currently 3 repositories:*(Imagine a "repository" as a "directory" or
"folder" similar to what you already have on your computer.)

1. https://github.com/Metagovernment/Community
<https://github.com/Metagovernment/Community>
2. https://github.com/Metagovernment/metagovernment.github.io
3. https://github.com/Metagovernment/ABC

You see the files/folder structure in each repository, followed by a "WIKI
PAGE" right beneath it.
That *"Wiki" *front page is stored in the file "*README.md*" which is
authored in the language called *"markdown"*
Inside that front page, you can *"link" *to other wiki pages authored in
markdown, which have to be contained inside a repository

To learn more about markdown, check for example:
=> https://stackedit.io/editor
=> http://markdown.de/ <http://markdown.de/>


*Now the interesting thing:*

How do you create a *"change proposal"* (=Pull Request)

1. Go to the "repository" that you want to make the change proposal to
(e.g. https://github.com/Metagovernment/Community)
2. Click the *FORK BUTTON *(upper right corner) => This will make a COPY of
the whole "repository" to your personal user account.
3. In your *repository copy*, click on the file you want to change (e.g.
README.md) and then click "EDIT" (a pencil symbol)
4. Author your changes in markdown and "SAVE" the file => which will
require you to write a small "commit message" and "description" of your
"CHANGE PROPOSAL"
5. Back on the main page of your *repository copy*, click *"PULL
REQUEST" *(it's
the only green button on that page) and make the pull request

*DONE :-)*

If you now go back to where you started:
https://github.com/Metagovernment/Community
You can click on PULL REQUESTS in the right side menu (a small "1" will
indicate that there is an open pull request)
It's here were all of us can discuss the Pull Request and once we agree of
merging it into our "Community Repository", we accept the pull request.

If you want to be "notified" about any activity around your favourite
"repositories", just click:
=> *Press WATCH*  (next to the fork button)
If you want to bookmark a repository and express that you like it
=> *Press STAR *(next to the fork button)
