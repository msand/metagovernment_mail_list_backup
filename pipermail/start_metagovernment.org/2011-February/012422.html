     BTW, before I forget:
     http://zelea.com/var/db/repo/votorola/file/4a948e370d4d/votorola/a/web/base/WEB-INF/web.xml
     That name (voff) won't work for the online pollserver (v), it'll
     hit the wrong database.  Better to fetch it at runtime like this:

        VOFaceWeb.contextPathToPollserverName(
          getServletContext().getContextPath() );

  2. Finish roughing out the bridge footings in the wiki.

     a) For a voter's draft, automatically highlight differences
        vs. candidate.

     b) Links from draft to bridge.  You'll click next to the
        highlighted text, and land at the corresponding diff segment
        in the draft.

     c) Indicate in the footed draft which user is on the other side
        of the diff (as you suggest), and link directly to his footed
        draft.

        And maybe:

     d) Stronger links from bridge to drafts.  The script runs faster
        than expected, so maybe we can do the same here: link from
        each diff segment in the bridge directly to the corresponding
        highlighted text.
