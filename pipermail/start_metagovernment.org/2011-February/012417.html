We finally have bridge footings in the wiki.  They're not particularly
useful yet because the interlinks (2,3) aren't coded.  But all the
hard parts have been coded.  Ed has a credit in the code:
http://zelea.com/project/votorola/a/diff/bridgeFooting.js

> >          (1)          (2)
> >
> >          -->          -->
> >    List       Bridge       Wiki

> >
> >          (4)          (3)

Demo: Start from the difference bridge.  For example:
http://u.zelea.com:8080/v/w/D?b=4044&a=4047&aR=2720&bR=2721
>From there you can traverse to the drafts:
http://metagovernment.org/w/index.php?oldid=2720&vodd=4047.-4044.2721
http://metagovernment.org/w/index.php?oldid=2721&vodd=4047.2720-4044.

  (you may need to refresh)

The text that differs is highlighted.  That's all, so far.

It also works in the pollwiki.  For example:
bridge: http://u.zelea.com:8080/v/w/D?b=3235&a=3355
drafts: http://u.zelea.com/mediawiki/index.php?oldid=3355&vodd=-3235
        http://u.zelea.com/mediawiki/index.php?oldid=3235&vodd=3355-

 3. Stronger links from bridge to drafts (2).  The script runs faster

    (plus any other stuff that's a priority over the theatre work)

CAVEATS

IE not yet supported.  Only tested on Firefox and Chrome.

Expect bugs.  In particular, you may see differences that fail to
highlight properly.  If you see any bugs at all, please post the
problem URL.  I'll fix them.

The footings are installed here:
http://metagovernment.org/wiki/MediaWiki:Common.js
If anything goes dreadfully wrong, an admin can comment out those
lines.
