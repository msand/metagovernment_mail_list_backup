We allow a source user (S) to "sign" a target user's page (T) in the
wiki.  This extends a trust edge along which authentication can flow:
http://u.zelea.com/w/Property:Truster

You don't have to be authenticated in the first place to extend trust,
but it's meaningless until you are.

We have no pretty graphs, just compiled data for the network traces:
http://u.zelea.com/system/host/u/home/v/votorola/out/votrace/
