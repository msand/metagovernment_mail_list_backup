>Where are you in development? In other words, what features are now
working, and what features/aspects are you still struggling to get to work?

I've stopped development a few months ago because i don't have enough
technical knowledge to continue it.

Actually, the platform is a mix of drupal modules and many of them are not
integrated or adapted.
I wrote about it on Edgeryders platform two months ago :
http://edgeryders.ppa.coe.int/spotlight-open-government/mission_case/d%C3%A9mopolitique-crowdsolving-tool
There are a lot of informations and links and technically, the only thing
that has evolved since is that a translation tool has been partly
integrated.

More briefly, the CMS is set to allow an adaptative and flexible database
of discussions with the use of folksonomy. I'd like to integrate
AlchemyAPI<http://www.alchemyapi.com/>,
a semantic analysis tool, but only to suggest connections to users.
This allows to link content by context and may avoid duplicate discussions.

I have not yet established an up-to-date list of features and aspects i'd
like to develop.
One of the latest tools i've started to design is a serious game described
in this doc :
https://docs.google.com/document/d/1OLzyVmNhwAU3E9-TSt7zFiUARSDh3pOetugUc7hFQ0w/edit
Its goal would be to facilitate a moneyless system where the "currency"
would be individual fullfilment.
