Dear reader, welcome.

This is a draft of a design document for Metascore, the software that
will power the Metagovernment project. You are invited and encouraged
to edit it, branch it, comment it, and send your patches to the
start at metagovernment.org mailing list for inclusion into the
repository. As you might have guessed, this document itself will be a
test of the kind of discussion that the designed software will allow.

This is not a text document, but a living thread of discussion
implemented through revision control. Therefore, it is essential to
grab the repository and not read a static checked out version. To grab
the repository, first install 'mercurial' from
http://www.selenic.com/mercurial/, then run:
--------------------------
hg clone http://freehg.org/u/SonOfLilit/metascore/
--------------------------



You can also read it at http://freehg.org/u/SonOfLilit/metascore/, but
please don't - instead, get mercurial, clone it, and participate in
the editing process -this process is the test of how writing
legislation in Metascore (according to my proposed design) will work!

 -- Aur
