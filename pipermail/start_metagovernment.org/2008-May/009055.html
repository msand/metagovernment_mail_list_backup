A real-time discussion does not need to be productive as much as to  
provide a milestone of sorts for us to measure ourselves by. I think a  
regular meeting (such as once a month) may give us some more focus and  
a better acquaintance with each other.

As for accomplishing things, if not by live meeting, then I am open to  
other suggestions. The items I see as needing the most attention are  
the ones listed on the proposed agenda:
http://www.metagovernment.org/wiki/Startup/Meetings/1/Agenda
We could move/copy that to another page and disassociate it with the  
live meeting... as long as it gets some attention.

I would like to get the project up to a point where we can start  
publicizing it, but before we can do that, I think we need a more  
consensual home page (ie, work on the wiki), some sort of project plan  
for Metascore, and some sort of bureaucratic organization such as a  
non-profit organization or at least a club charter.
