After you install mercurial, you can use it to clone my repository to
a local copy

hg clone http://freehg.org/[...]

then you enter the folder it created

cd metascore

Then you can read and edit anything in there like it were regular files.

Once in a while you commit:

hg commit -m "quick list of what you changed"

you can then export a patch with your changes and send it to the list

hg export -o patch.diff

or you can create a repository in freehg.org and there are
instructions there on how to use it.

to pull in new changes:

hg pull
hg merge

google for further assistance or ask me.
