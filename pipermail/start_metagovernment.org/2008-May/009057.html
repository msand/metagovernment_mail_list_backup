You need to specify which changset you want to export. If you just want to
export the last commit, you can type

# hg export -o patch.diff tip

"tip" being a mnemonic for the latest change. Otherwise you have to specify
the changeset id, which you can figure out with

# hg log

Personally I would recommend, you simply create a repository on freehg.org,
push your changes to that, and let aur decide, if he wants your changes or
not.
