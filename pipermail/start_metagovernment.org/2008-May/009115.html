I prefer the creative commons one. Not because it is substantively different from GNU (which wikipedia use) but because the whole raison d'etre of the CC project is to make maintain a set of the licences for exactly this sort of purpose. They have a very short and snappy "human readable" version http://creativecommons.org/licenses/by/3.0/ which links thru to the in-depth legalese version. All of this is similar to GNU, I just think it's been done a little slicker. It is intended towards "cultural projects" such as books, wikis and blogs rather than software (perhapd GNU is better at software? - although I've heard some debates around backwards compativility of licences and other niggles with GNU). It is also a major international movement and a project we can learn from, support, and perhaps down the line get some support from.
