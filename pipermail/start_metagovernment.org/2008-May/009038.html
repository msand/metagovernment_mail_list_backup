I can hardly see how you would participate in a discussion or edit
legislation effectively on an iPod. You could use that kind of device
for reading. For writing, the minimum hardware you need is a
keyboard, a screen and a network connection. This may be satisfied by as
little as a blackberry.

You need a certain level of technical infrastructure to run metascore,
the question is, where you draw the line. I know we're all out to
conquer the world here, but lets start with something for ourselves and
move up the ladder.
