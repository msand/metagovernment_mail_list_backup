I don't think you are alone, Michael. A lot of people prefer a list  
server. One primary reason is that it is push technology, while a  
forum is pull. That is, e-mail comes to you as it happens, while a  
forum message is only noticeable when you seek it out.

Invariably some people will prefer a forum and some a list server.  
We're about synthesis. I think we can make both work.

I think I can simplify the requirements a lot. I have had major  
reservations about phpBB because I have used it in the past and found  
it to be extremely primitive. But today I had some time to give  
version 3 a try, and it appears to be a lot better than earlier  
versions.

If we implement phpBB 3, then there is only one programming task, plus  
some fairly straightforward technical work.

The programming task would be to get M2F to be stable with phpBB 3. It  
is designed with phpBB 3 in mind, so that hopefully is not a huge task.

If that can be accomplished, then there only remains the technical  
setup, which is:
1. Set up phpBB 3 on the Tuxfamily site, in /forum or in  
forum.metagovernment.org
2. Install and test OpenID mod:
http://sourceforge.net/projects/phpbb-openid/
3. Install and test completed version of M2F.

I can do steps one and two, but my time is still very limited, and a  
task like that takes a few hours of dedication. I am not sure when I  
could get to it, but I can open up an account on the tuxfamily site  
for someone who does want to do it.

If we do go ahead with this, does anyone have any input on the forum  
areas I suggested in:
http://www.metagovernment.org/wiki/Forum_integration#Forum_topic_areas

With subtopics allowed in phpBB, I would suggest:
- Democracy and Governance (philosophy)
- Active Projects (introductions and discussions of particular projects)
- Technical Discussions (e.g., of crossforum)
- Metagovernment Administration
- General Chat
   - New User Introductions

One thing I haven't figured out about forum/list integration: how does  
the bridge know which forum to post new threads to? That is, if  
someone starts a new topic on the list, where does it get filed in the  
forum? My guess would be that everything goes into one category (such  
as General Chat), and then an admin can re-file them as needed.
