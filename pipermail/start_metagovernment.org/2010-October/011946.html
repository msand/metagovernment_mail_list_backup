Hi Ed,
I have to say I have been very unimpressed by how this question went.

Probably we are here looking at another intrinsic limit of Vilfredo.
Although the question is in theory open, is not really open. There is
only a certain number of adjective that can be used, so essentially it
is bound by those adjectives, and thus at the end when we see we
cannot reach a consensus on those we try to wiggle out on the meta
answers.

If not we can just go back in the history, write down ALL the names
that have been used, and then use a Condorcet voting system to decide
among them. That would give us an answer.

Maybe we should start a meta question: how should we ask about the
name of the kind of democracy we want to develop?

But also I am not sure if we all agree on what kind of democracy we
want to develop. So my heart is with the meta answer that says to
redefine all the terms, and then use that definitions in our
discourse. This because it is a plan that open doors, not that shuts
them. If you see what I mean.

I think if we continue to wrestle eventually we will all give up,
either not voting (which I don't like), or by all voting for an escape
strategy. Something like: We agree to disagree.
Or we are not yet at the time to agree,
and then we move on.
