I would go as far as to say that any auto filtering is inherently biased to
the person or group it was created from. Even, if the group is a large mass
of people, or even the majority.

I think the task is to allow the user to more quickly sift through the
information efficiently. Then he can take his own side on the issue on a
more educated level. I think Settleit is tackling that issue pretty well
actually. Now we just need to observe how it works out and learn from it.

Cheers,
James

>>
>> There are several other methods being worked on to discourage trolls.
>> Stackexchange has made some progress by creating clear and specific
>> community. The National Communication Association has been researching this
>> http://engagingnewsproject.org/social-media-buttons/. Many people are
>> working on it from several different angles. The SettleIt.org design
>> reduces the cognitive load of any single interaction by isolating it so you
>> only need to consider a single statement at a time which also allows us to
>> divide up the work among multiple people. This allows for the time and
>> effort you mentioned in other options of reducing trolls.
>>
>>
>> "OK, so every SettleIt debate has the option of including a branch
>>> which will settle the issue of whether the Bible is true ;-)  Maybe
>>> now you can see why filtering is a good strategy?"
>>
>>
>> Yes, many of the statements will include a debate on the Bible. I don't
>> see how that justifies filtering it out. In SettleIt.org you would be able
>> to have the same branch in multiple areas so as that debate is worked out
>> it will automatically be updated in all the other affected areas. To
>> dismiss someone's concern is to exclude them (and yourself) from the debate
>> which defeats the purpose of the system which is to build agreement. To
>> allow an individual to filter what they see builds bias into the system
>> further separating people into polar groups which again defeats the main
>> purpose. Since SettleIt.org is a tree structure you just need to see the
>> statement once in each affected context and you can decide to expand that
>> statement to see the debate. Otherwise it is very small a distraction which
>> provides value in understanding the concerns of the group.
>>
>>
>> "I'm not denying any of them their right to free speech,"
>>
>>  It's not a free speech issue.
>>
>>
>> "Some people are just going to have to go along with what the majority
>>> (or supermajority) prefer."
>>
>> Although that statement is true, in many debates you can not reach even a
>> simple majority because many people are dismissing concerns of the others
>> and refusing to debate the topics. I don't see why you would discourage
>> efforts to increase the level of agreement.
>>
>>
>> "Seems to me it's just more efficient to flag something as "unsupported
>>> opinion" and take a few percent off the weight for every person who
