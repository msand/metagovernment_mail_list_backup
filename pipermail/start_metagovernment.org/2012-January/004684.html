Hi,

since i have now joined the start list and also had some time to review
parts of the wiki and other material, let me do a quick recap of the main
points of my original message and re-state my proposal/request:

My name is George Anadiotis and i am a researcher and software systems
architect with a keen interest and background in distribution,
self-governance and social media, all of which i believe are part of what
is termed metagovernance. You can find some information on me and my work
on my blog <http://ganadiotis.wordpress.com/> as well as on my Metagovernance
user page <http://metagovernment.org/wiki/User:Ganadiotis>.

Recently i applied for a lightning talk slot for the upcoming
fosdem<http://fosdem.org/2012/call-for-lightningtalks>, providing the
text published on my
blog<http://ganadiotis.wordpress.com/the-peer-to-peer-work-group-toolkit-project/>and
on
MixedInk <http://www.mixedink.com/#/DecisionMakingP2PFOSS> as a description
of the talk. The page was set up rather hastily in order to meet the fosdem
application deadline, so unfortunately i did not research as thoroughly as
i could have - otherwise i would have found this group beforehand. Since
however i agree with and endorse the goals and principles of the group as
laid out on the wiki, and since my talk is in line with these as well, i
think it would make sense to:

1) Inject parts of http://metagovernment.org/wiki/Presentation/Generic in
the talk description
2) Add the talk description in http://metagovernment.org/wiki/Presentation
3) Add a link to the above in the talk description and give the talk -if
granted the slot- as a member of the Metagovernment group, presenting it to
the fosdem audience

If there is consensus on this, i would like to do this asap since fosdem
lightning talk applications are currently being evaluated.

regards
George Anadiotis
