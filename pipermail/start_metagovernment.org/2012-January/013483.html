I looked at it, and it seems rather heavy-weight in terms of system resources (because it is Flash) and yet somewhat light on collaboration tools. The whiteboard is nice for drawing but not very good for collaborative text.

Instead I would like to use PiratePad, which appears to have undergone some upgrades that makes it even better to use than before. Most notably, the chat window can be re-sized. So I suggest that we hold the chat part here:
http://piratepad.net/clKyRYmtD0
However, if others prefer tinychat, I'm OK with that of course.
