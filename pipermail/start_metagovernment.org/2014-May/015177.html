I believe the system only allows for one proposal to be considered at a
time. Anyone can propose a new proposal if one is not already there. I
believe everyone can even close the current proposal and then create a new
one. I can see no way to edit a proposal so when better wording was
suggested I close the current proposal and create a new one with the new
wording. Since I was the only person that voted for the proposal and the
updated wording was clearly superior I didn't think anyone would mind the
proposal changes.

Theoretically you can tell what proposal the comments applied to as the
proposals and the comments are listed chronologically in the discussion
text. It's not easy to see though as the textual changes are not fully
shown in the list.
