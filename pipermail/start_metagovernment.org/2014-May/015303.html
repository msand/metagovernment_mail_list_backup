I've no problem with professional courtesy and definitely empathize
with his situation (I've got an ailing mother myself and am currently

the home page is already down, let's go ahead and finish it with the
implied consensus that when Ed (or whoever) wants to revisit it we
will do that.

So, back to the big question, what is *your* current vision of what
this project is about?
