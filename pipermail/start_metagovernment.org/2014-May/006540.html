Stepping up one more level of meta on proposals within this community… we really need a uniform, easy-to-understand way to track and understand proposals within this community.

For example, Steve, your latest proposal is very well-thought-out, but the subject line leaves a lot to be desired. :)

So I propose each proposal have a unique number, be presented in a uniform format, and be tracked in a centralized place:
http://metagovernment.org/wiki/Proposals

Instructions are on that page. This too is a proposal, so following this new format, I called it PROPOSAL 1. If you agree with this simple, helpful format, please re-introduce any active proposals using this format so that any onlooker can understand what is going on.
