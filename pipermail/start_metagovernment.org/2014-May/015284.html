+1 to Ed.

I have been away for medical issues of my own so I haven't been able to
contribute. But here is my two cents here.

In my experience as a web developer you never put up the "under
construction" sign. As much as we would like to think we know what will
happen we don't. The last thing we want is an extended blankness for some
reason, or people have no idea what we do and perhaps lose membership.

If we want to discuss what we want there and then replace when we actually
have something that would make more sense.

I've been doing web consulting for a while so that is IMNotSoHO.
