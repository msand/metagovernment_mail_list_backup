Writing an app that works directly on FB has been rather daunting, 
largely because the platform changes so quickly and the available 
documentation is relatively disorganized and confusing.

If you want to write a serious app for FB, it helps to have good 
familiarity with PHP and a backend db (I use mySQL). You'll also need to 
become familiar with their html and js extensions, FBML and FBJS.

On the other, hand, it's also possible to build your own site with your 
favorite tools and just use their FBConnect tool for authentication.
