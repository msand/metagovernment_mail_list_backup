This version relies on Facebook authentication to facilitate 
one-person/one-vote participation. An earlier version (still running at 
ChoiceRanker.com) had its own login system, but that took a lot effort 
to build and maintain, and left me no way of leveraging the social web.

As a result of that experience, going forward, I'm committed to using 
existing social authentication platforms for login so that I can focus 
on building tools for content collection, comment, coalition-finding, etc.
