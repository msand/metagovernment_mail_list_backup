*local* regulation, such as a municipal code or zoning bylaw, by

US State level voting won't be possible (w/ Votorola) until someone
designs and codes a synchronization interface for the voter register.

combining the smaller lists of the individual pollservers in each
city/region into a single aggregate list.  I'm not working on that
myself (job is open).
