Valid registered trademarks beat domain names but a domain name can give
you a valid registered trademark.

If your code is to be GLPv3, as,an example, then your text can reasonably
be some version of CC.
After you figure out which version then it might be possible to think about
your art work and graphics. The head scratcher might be patents.

Do you understand the different versions of CC, not all of which are
"free"?  You might chose CC with derivatives and attribution.
