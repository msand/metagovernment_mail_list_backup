Surely the auto-match system needs to be a little more complicated than
signing up to Facebook.

If it's a psychological match you'd presumably need to ask a lot of
questions to get some variety in the matches, and not have determined boxes
we fall into. How would these questions be chosen?

Are you assigned a different match for each vote dependent on the closest
match you have who bothers to cast?

Would users have a 'block future matches' button if they didn't agree with
the decisions that their match chooses?

I think I'm in agreement with this strategy. But the auto-match
system would have to be comprehensive and verifiable and so I worry others
might then consider it too invasive and not bother filling out a survey at
all.
