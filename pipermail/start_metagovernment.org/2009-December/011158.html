Dear Ed Pastore,
in the discussion
How should we handle the "wall of text" problem?
http://vilfredo.org/viewquestion.php?q=67&room=Vilfredo

you seem to be in  a key position to produce a really helpful new proposal.

You did not support the proposal n.1304, which starts with:

"Forcing people to write a certain way is the wrong approach, but good
writing should be encouraged as much as possible. There are three
(maybe more) approaches to doing this:

   1. Give guidelines about good writing.
   2. Allow others to give feedback about the writing.
   3. Give immediate automated feedback about the quality of the writing.

..."


Complete proposal available here:
http://vilfredo.org/viewproposal.php?p=1304

Yet if you were to support it, the pareto front would shrink of one element.

How would you rephrase that proposal in a way that would permit you to
support it?
Possibly in a way that would still let the people who supported it
before still support it.

Could you please write such new version in the question page.
http://vilfredo.org/viewquestion.php?q=67&room=Vilfredo

If you think this is not possible, it would be really helpful if you
could write instead a comment
(Starting with a line saying just COMMENT, in bold)
explaining why you think the above proposal is not just unacceptable,
but unacceptable in any possible variation.
