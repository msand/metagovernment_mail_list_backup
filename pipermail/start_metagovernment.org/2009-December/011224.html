Hello folks,

after talking about the open network of voting tools way to long (mostly 
on the Votorola-List), we actually started coding it. And it actually 
seems to be really easy. It took only 2-3 days to code a 
prototype-mirror to pull the votes of the Nationbuilder-Sites (USA, 
Canada, Australia) over to Votorola. That means: now you can vote in any 
of these 3 tools and your vote will correctly be displayed in Votorola. 
(It takes about 2-6 hours for the transfer.)

Here is Votorolas poll: 
http://t.zelea.com:8080/v/w/Pollspace/?u=X-natP-99-ZeleaCom&p=edor

The red dots right in front of the User/Position names indicate pulled 
votes and lead to the external positions (the black ones lead to 
internal positions).

These are the sites we mirror the votes from:

http://www.whitehouse2.org/priorities/2365
http://parliament2.ca/priorities/99
http://au.nationbuilder.com/priorities/72

Of course by now the mirroring only works one way. Therefore we invite 
every other project which is interested in an open voting-network to 
pull Votorolas votes too. Let´s connect! ;-)

We publish our votes here: http://zelea.com/var/tmp-public/ ^[1]

Until we have a common registration (about next week) we register 
manually. So if you want to try out the Votorola-Mirror right now, 
please send us your user number of WH2 for example and we register you. 
Jim Gilliam and the Canadian Administrator are already added.

Have fun!

Greetings, Thomas
