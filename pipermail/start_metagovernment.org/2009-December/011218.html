Hi

I would recommend taking a look at how http://www.kickstarter.com/
works. I think this model is a good approach. The current drawback of
kickstarter is that it only works in the US because it relies on the
Amazon payment system.

This is ideal for funding things like additional features but would
not work for things like funding ongoing server costs.
