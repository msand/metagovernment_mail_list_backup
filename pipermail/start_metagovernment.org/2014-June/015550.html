I thought again.
I oppose the draft as is and make an alternative proposal instead...

*PROPOSAL:*
If i think of the Front Page or about Metagovernment as "One Node in a
Network", then those Sections make sense.
"Framing" is the way of PRESENTATION (includes themes, styling, layout, ...
and so on.
"Body" is the CONTENT of the Node in the Network
"Linkage" is about all the CONNECTIONS the Node makes to any others in the
Network
"Administration" is about the collaborative GOVERNANCE, thus how the
CONTENT, PRESENTATION and CONNECTIONS of that node we are talking about can
be changed by people.

So i would stay with those 4, but i would rename them to:
"Presentation, Content, Connections, Governance"
