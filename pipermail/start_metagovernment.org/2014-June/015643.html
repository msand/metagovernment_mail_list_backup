Alex.  I found information and read it in five minutes.

Here is my take.  Etherum and maidsafe look a lot like freenet.  Both
involve making money in exchange for administration.  Etherum is
distributed agents using a protocol on top of freenet like.  But really it
is Bit coin infrastructure too.

Maidsafe is just freenet concepts..  Think Kim dot com.

Doa is just a abstraction of persistence.  Might be a good idea.  Do you
want to bend over for Oracle or for Google or for microsoft? There is
something for c++ though.

Otherwise, it will tend to be inefficient because it is yet other
abstraction layer.

Etherum is the best.  But agents were big last millenia.  Not here yet.
You would like it because the agent could vote for you.  And so i suspect
shilling of a  sort here.
