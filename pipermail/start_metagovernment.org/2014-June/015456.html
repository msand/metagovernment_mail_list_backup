Hi Bentley,
for Vilfredo to work properly you have to tackle the whole question all at
once. Not part of it. This because the domination process (the way a
proposal dominates and kills another proposal) should only happen between
two alternative proposals that solves the same problem.

So if we decide that the structure should be "mission", "vision", ... Then
we can open a separate question for each of them. Running parts of the
proposal inside only confuses everything, and can risk of deleting
important information. You run the risk of finding a small element that
everybody agrees withm and reaches consensus. But does not solves the
question.

Thanks for asking,
