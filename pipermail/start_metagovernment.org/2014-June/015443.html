Hi Mike,
I am trying to understand.

So you felt that Vilfredo was not part of the decision process. But was
some type of "brainstorming". While me, Ned and Bentley felt this was the
actual decision going on.

So you made a proposal which from your point of view was independent, while
for us it was trying to shortcut a decision system that was being
implemented. When we voted against it (and in a way you did not expect) you
felt that your proposal was unjustly struck down. and this felt that we
were "forcing on you" Vilfredo.

So they key element is that according to you Vilfredo is a brainstorming
method while according to me, and Ned and Bentley is an actual decision
mechanism.

And the other key element is that we did not "vote on what system to use".
I made an official proposal to use Vilfredo. Steve asked how to proceed,
and Ned did not wait for the voting process, but simply started.

If this proposal to use Vilfredo was gone through the vote (so a vote on
the voting system, a meta-vote), then we would be ok in using Vilfredo, and
you would not feel forced anymore.

Does this feel like a correct description of the situation?
