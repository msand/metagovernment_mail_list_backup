While we are clarifying with Michael Allen, Vilfredo is going on.
Unfortunately Vilfredo cannot really be stopped. At maximum Ned can wait
for the maximum time (2 days instead of 1) to move on the question.

So my suggestion is that we keep on working with Vilfredo and use this as a
case study. A tutorial. Then maybe we run an official proposal to see if
there is a wide enough majority to implement the result on the homepage.

http://vilfredo.org/viewquestion.php?q=164

The alternative seem to me would be not to use it anymore for this
proposal, but this would throw away all the work done.

Right now there are 4 people who have written new proposals. 4 proposals
inherited from the previous generation, and 20 new proposals.

It will be a long voting session ;-)
