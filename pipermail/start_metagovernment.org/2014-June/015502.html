Sure we can.  We agreed to blank the old home page because it "solves
the problem of false and misleading information, while leaving the
door open to further proposals and content."
http://metagovernment.org/pipermail/start_metagovernment.org/2014-May/006490.html

We didn't impose any restriction on further proposals, nor on the
content we can add.  Presumeably we won't repeat the mistake of adding
"false and misleading" content, that's all.

To be sure, that old content was nobody's fault.  It was the result of
an early experiment with a half-finished consensus tool, which (to
make matters worse) was misused as a decision tool.  It left us stuck
with a monster of a home page.  Now if we repeat the mistake and again
force a decision through an experimental tool, then we risk another
Frankenstein monster.  *That* would be like an end-run around the
blanking decision.

We should experiment, but should also protect the integrity of the
decision process... which we're doing.  So there's no problem here.

No longer blocking,

Mike
