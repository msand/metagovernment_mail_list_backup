Ed and Ned agree that proposals should be implemented only if there are no unresolved objections to them.

Ed tentatively defines a proposal as having unresolved objections when there are any significant number of dissenters as defined by a percentage of the total number of people voting on that proposal.

Ned defines unresolved objections as any objection which is defensible in discourse. And the standard of "defensible in discourse" is not inherently subjective (and thus arbitrary and thus authoritarian) because _________.
