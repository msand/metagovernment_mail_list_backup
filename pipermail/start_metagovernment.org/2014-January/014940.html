I’d like to reiterate Michael’s question and encourage others to answer it:
> What would it take for you to participate in Metagov even if it did *not* have a mailing list attached?  What would you require in order to participate even when it was not directly in your own interest, but rather for the sake of the group?

Also, I agree with Reid that we should experiment more. That’s a long tradition here: experimenting on our own community to see if we can find models that can be reproduced in other communities.

I think we can turn the weakness of this list server into a strength in that regard. How about if we start trying out other systems for set periods of time, but simultaneously we keep this list open as a sort of meta-conversation about what we’re trying within that system. Note, we have effectively done this in the past a little bit with Vilfredo and with Votorola:
http://vilfredo.org/viewquestions.php?room=Metagovernment
http://u.zelea.com/w/G/p/m00
In those instances, we just grabbed a tool that looked handy. I’d be up for deliberate experiments where we try something for a set period.

Also, Marc, back in November you hinted you’d try to look into the Pirate Party’s tech for linking a list server to a forum. Any progress on that? :)
