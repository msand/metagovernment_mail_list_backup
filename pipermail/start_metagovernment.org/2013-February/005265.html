I've installed Asirra:
https://www.mediawiki.org/wiki/Extension:Asirra
which selects photos of cats and dogs from Petfinder and makes you choose the cats to proceed. Very cute, and perhaps effective (though the page reports that bots may be cracking it). I changed the default pics per row to see if that confuses bots further. I reenabled URLs momentarily to test, and it appears to be working. I turned off urls again for now until we see how it works. If it is effective, it should stop the bots from creating their new user accounts. If we so no new bogus users for a few days, let's try turning URLs back on.
