Following your lead, I'm testing a captcha for the pollwiki:
http://www.mediawiki.org/wiki/Extension_talk:QuestyCaptcha
This one allows for context-dependent questions, which I figure are
harder to out-source to other sites.  E.g.:

   This site's domain name contains a five-letter word.
   What is the word?

   At the top left corner of this page are symbols with spikes
   sticking out of them (chaos symbols).  Count all the spikes on all
   the chaos symbols.  How many spikes are there?

   As the main page explains, ours is a special kind of wiki.
   What kind of wiki is it?

I only set the last one.  See if it stops the spambot registrations:
http://zelea.com/mediawiki/index.php?title=Special:RecentChanges&limit=500
