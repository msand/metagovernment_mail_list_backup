Sorry about that folks... just as with "walls of text," I have noticed  
that e-mails with a lot of links and/or with a lot of different  
options tend to be conversation-killers. I did both below, so allow me  
to simplify:

How about if we use PiratePad instead? It is identical, except it is  
run by the Pirate Party instead of a for-profit company:
http://piratepad.net/fiuASvRqss
There is no explanation, but hopefully that also means it does not  
have the 16-user limit.

My only objection is that the text window is rather small. If  
conversation gets lively, then posts might fly off the screen before  
people can even read them. Does anyone have a solution for this?
