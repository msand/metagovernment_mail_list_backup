Welcome to the list! Good to have you aboard!

Yes, much has been done in this field already. Quite a few, sometimes 
very similiar, tools do already exist and it seems like every month a 
new one is emerging. Good, that you did a search before starting to 
really work on it. ;-)

What I find most interesting in this field at the moment is the picture 
of all of these tools working together. Not competing for voters (which 
would be a setback for the e-dem-movement), only competing for users. 
And it seems possible, if we work out a way to have a common 
registration and then mirror the votes (and proposals) from one tool to 
all the others.

I fully support Mikes proposal here: 
http://t.zelea.com/wiki/User:Mike-ZeleaCom/Opening_an_architecture

What do you think?
