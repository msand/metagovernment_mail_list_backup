> The -10 to +10 style of voting you describe is called range voting. Ranked
> voting is ordinal.. 1st, 2nd 3rd, etc.
>

That is absolutely correct.
The principle behind vilfredo can work with any of those kind of
voting. It could even work with each person choosing to vote in a
different way. Provided that each person is consistent with itself in
one generation (that is each generation, for each question, each user
can chose how to vote. But no changing in between the same
generation). This is because essentially all the algorithm does is
extract a partial ordering from the votes, for each user, and then
just use the partial ordering to decide what should remain.
