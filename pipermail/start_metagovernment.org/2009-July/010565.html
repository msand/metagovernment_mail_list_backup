The GNU Affero License does not require the particular use of any
repository. The only requirement is to link the source code of your
running code to the users of your web application, just like the normal
GPL does for desktop apps.

As for the http://bitbucket.org/plans/ i don't know, but if there is any
problem using that you can also use any of these free popular Open
Source repositories:
- http://sourceforge.net/
- http://savannah.gnu.org/
- https://launchpad.net/

note: code.google.com hosts several projects too, but they do not allow
Affero GPL licenses... guess why ?  :-)
