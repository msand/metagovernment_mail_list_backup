Hi Mike and Ed, (Great Responses -- much appreciated!)

So Mike, if a proposal constitutes a vote, is it also true that pretty 
much any speech act of rational discourse constitutes a vote of some kind?

If proposals constitute votes, then discourse always has voting built 
into it, so that discourse can be decisive. Am I missing something?

If we add Ed's (to me) clearly valid suggestion (that the spectators of 
a discourse, if they are free to participate, are voting when they do 
not participate), then again, public discourse always has voting built 
into it, and can be decisive. Again, am I missing something? Why would 
you assert that discourse cannot be decisive, if discourse has voting 
intrinsically embedded in it?

If we use your definition of voting, and if voting is, as you suggest, 
built into discourse, then the distinction I have been trying to express 
perhaps becomes that between "naked voting" (votes cast outside of 
discourse) and "discourse voting" (votes cast through discourse). 
Votorola uses "naked voting". Blinap uses only "discourse voting".

My contention is that we would be better off to use only discourse 
voting in our (decisive) public decision systems, in order to bring to 
bear upon democratic decision making the values of discourse that 
Habermas talks about. Naked voting does not bring those values to bear.
