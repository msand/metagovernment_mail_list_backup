Thanks for the advice Pedro! Actually this is something where we don't
have a clear opinion, so it would be great some help to clarify it.

On one side we want our tool to be free software, so anybody can use
it, mix it, improve it, etc. But on the other side, our main purpose
is that such a tool exist, independently if it is our tool or the one
of some company, and the people is able to use it and change the world
with them. The same I'm using gmail now, and it helps me a lot for
good stuff, but I still have free or not controlled by companies
options for having e-mail.
In that sense we thought GPLv3 was good (not so free as a BSD license
for example, but not so restrictive as the AGPL), because it is free
software, but allow some "freedom of not using it as a free tool" so
it could be easier to spread it (even through companies that we don't
like), and finally have some cool tools to debate much better and
useful than the ones we have now (that is our main concern).

What do you think?
