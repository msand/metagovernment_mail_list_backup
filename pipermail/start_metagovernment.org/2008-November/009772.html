I like this idea, though would suggest we make it less formalized in  
order to allow Metascore to scale up and down different sized  
communities. In other words, maybe we should allow for discreet levels  
of approval without fixing them either by percentage or by name. So we  
would just have something like level 1, level 2, and level 3; and  
allow each community to set the particulars.

And as an aside, we should be moving away from talking about laws. On  
the main page, the words "resolution" and "law" have been mostly  
replaced by "proposal" and "policy." This again is to allow for  
scaling beyond the scope of governments. Chess clubs don't make laws,  
but they can certainly have policies.
