On Nov 5, 2008, at 2:42 PM, Jacki Buros wrote:

> It would be interesting to put together an initiative to greatly  
> expand local governments, putting most if not all bills up for a  
> (non-binding) popular consensus vote.. This might be. a first step  
> in guaging the enthusiasm at large for such a form of governance.

I think the general thought among this group is that we should start  
with even smaller groups... i.e., ourselves and other small startup  
communities. From there, we can learn how to fine-tune the software:  
things like what (mathematically) constitutes a consensus, how scores  
should be weighted, whether or not users should have scores, etc. It  
seems that small-scale, real-world tests would be the best way to  
learn true answers to these questions.

P.S. Note Manuel's prototype is almost to what I guess would be the  
Alpha stage, where it will be ready to act as the tentative governance  
mechanism of the Metascore project, if nothing else.
