I have looked at all the projects on metagov's site as well as a few more
such as civitas <https://www.cs.cornell.edu/projects/civitas/>. Shockingly
i have not been unable to find a single platform that satisfies
my requirements, which I believe are pretty basic. See below.

*Uses (what is the platform designed to be used for)

   - Any type of voting from small groups to the worlds population


Requirements

   - Anonymous as well as non-anonymous voting
   - Verifiable voting (users can check their vote counted)
   - Delegated voting (allowing users to subscribe and create parties)
   - Modular (allowing developers and server admins to add or remove
   features)
   - Verified voters (must verify that a voter exists, has only voted once,
   and can vote for that election)
   - Support for multiple voting methods such as Schulze and Ranked pairs
   - Web and mobile
   - Extremely secure. Designed to be protected from all categories of
   attacks.
   - Multi-lingual

*
