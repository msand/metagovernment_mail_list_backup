Yes, that's part of the reason that the "violence" reference is in there.
Also, when I talk about the project with people, some immediately jump to
the conclusion that we're advocating some sort of revolution. I think there
needs to be something in there to indicate that we're focused on peaceful,
consensual change.

 

Concerning refinements to the text overall: I think those will work out when
we try to compile it into a video. The wording sounds somewhat formal and
stilted to me. I think as we say it aloud and associate it with images, it
will evolve.

 

Which brings me back to that question; can folks please help come up with
the images that would go along with this script?

 

What might be the best would be if we had an animation which depicted the
ideas from the first-person perspective of a character. But failing that, it
could be a sequence of images. Or... whatever ideas any of you might
have...?
