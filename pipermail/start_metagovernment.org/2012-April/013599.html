> Concerning decentralization, what role does [zelea] play in this?
> The poll button leads to http://zelea.com/w/G/p/m00
> Is that simply another configurable point as any other?

That's where the poll is defined.  A poll is always defined in a
pollwiki (top left): http://zelea.com/project/outcast/_overview.xht

Metagov's wiki is not a pollwiki, so the link is external.  A given
Metagov draft can (in principle) be defined in any pollwiki, but in
practice we don't support multiple pollwikis yet; we still have some
work to do there.  All we have is the reference pollwiki for Outcast,
which is currently running under my personal domain name (zelea.com).
