There are so many difficulties, maybe we can all brainstorm what of them we
can even identify, then go about identifying what
solutions/remedies/strategies we have available to handle them? Is anyone
else interested in trying this? I don't mind starting a prefab wiki we
could use to do this on, or if someone wants to use bettermeans.com or
what.

I do wish to address this collectively however because I already know one
monumental issue the world square project is up against: what's going to
draw even 10% of the world's people there? How will it serve them? How will
it handle it's disruptors? (people organizing to stop corporate greed will
face corporate backed windbags who can bribe moderators) how will anyone
know it isn't corrupt?
All these questions I ask with constructive intentions, and to each one I
have a number of my own favorite strategies. but this is the commons so
lets work together. Anyone?
