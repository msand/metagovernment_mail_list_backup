I couldn't find a way to add to Fabe's suggestion or even put the
suggestion as another entry, so I will mention that I completely agree
with the suggestion starting as "I think we need to focus to the
*wiki* as an equal to the *mailing list*".   See also [[Edit policy]]
for a related discussion.

The wiki concept supports the idea of multiple entry points into the
wiki space.  The main page can be an "official" entry point for
relatively static content, but there ought to be a "News Reel" page
that lists "Most active pages" or "Hot topics" -- Pages on the site
which are showing activity, debate, discussion to keep part of the
site in user-time for those who frequently check-in to be part of the
community.  This could (and SHOULD) be an automatically generated page
so as not to create much additional work to create, and even less to
maintain.

I have a much more elaborate (and ultimately, I think, more
productive) solution to this problem but it would involve some pretty
hefty development of the mediawiki software to allow voting (thumbs
up/down style) of each revision, or something of that sort.  Some
loose comments about this are at [[User:StalkingTime]].
