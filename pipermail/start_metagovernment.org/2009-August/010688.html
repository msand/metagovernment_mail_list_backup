If anyone has made an invitation to another group, please denote it on  
the Related projects page.
http://metagovernment.org/wiki/Related_projects

If any group joins, please denote it on the Active Projects page (and  
the Main page):
http://metagovernment.org/wiki/Active_projects

I have started adding dates to the invitations I have sent. If  
anyone's invitation is not successful after a week, someone else  
should send a follow-up invitation. The advantage of this is that the  
second person may choose a different contact style and/or in any event  
the recipient will know that there are multiple people inviting them.

I would encourage everyone here to send at least one invitation. It  
takes very little effort.

Thanks.
