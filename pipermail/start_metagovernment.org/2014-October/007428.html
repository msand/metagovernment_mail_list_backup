Some news from the Vilfredo front.

We employed as a consultant Giovani Spagnolo, (who some of you might
remember), and things started moving faster. We chose together github as
the development tool. Using in particular the issues there to track bugs
and suggestions.

Now things has started moving so I feel I should let you all know. Many of
you have asked how can you help, well there you can. We need programmers,
Designers, testers, fund raisers, people to spread the word...

We will also soon be developing the API to permit to Vilfredo to speak with
other services, so others can open a question, and then when the question
has ended receive the answer.

The question to go is at:
https://github.com/fairdemocracy/vilfredo-core/issues
https://github.com/fairdemocracy/vilfredo-client/issues

Also many of you have tested Vilfredo recently. If you have any idea of
improvement that is the place to go.


Cheers,
