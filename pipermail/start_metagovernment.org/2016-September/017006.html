I'm thinking it would not be a voting process, but a vetoing process. These
proposals are assumed to be created by the experts, only significant
opposition should prevent it from passing. As an example, a proposal would
need to double the number of vetoes acquired from the general public than
it initially got votes for in the committee stage. The proposal has the
ability to be reintroduced if the vetos for the veto doubled the number
again in the general public. This, in my view, removes SDAP's ability, they
aren't able to push things through easily. Education would play a bigger
role.
