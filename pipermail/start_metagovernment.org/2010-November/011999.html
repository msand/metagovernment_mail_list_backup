good points. Nevertheless, if we have learned one thing out of those

any lawful objective *they* choose, wherever and whenever *they* choose.]

In a democracy I understand people who refuse to vote on something,
when all the options have been chosen by others, and they had no way
to write their own. But in all the project we have here in
metagovernment (and for sure in Vilfredo) you are allowed to write
your own options. More than that, on Vilfredo we make sure this is
done in a fair way by not permitting anyone to vote (or read) anything
before time.

[I commend you for trying to make Vilfredo as flexible as possible.
However, as I am sure you know, you still have a long way to go in that
regard. In particular, no one should be required to use your software or
anyone else's software to document their own goals and objectives.  I
believe we can do far better than that, e.g., by using the StratML standard
to enable individuals to document their own goals/objectives anywhere on the
Web and building applications and services leveraging the data contained in
those "plans".]

If you had such a strong position on something, would it not have made
sense to write it, so that we could all have voted it, instead of
