Ok, a basic overview could be this:

We WONT develop an application for participation, only voting and delegated
voting of propossal that will come from other political parties.

The project is in a early stage. We have been studying the feasibility of
building a secure voting system with delegation. It seem it is posible, so
now we will build a voting application which fits the spanish voting
requirements, at three levels: local, regional and national.

We are using the e-DNI document; this is an oficial smartcard emited by
national government that we will use for autentication purposes.

David Ruescas, who is on this list too, is in charge of developing the
security scheme, and he could write more detailed information. As far as I
know, the security scheme was first based on Adder:
http://freshmeat.net/projects/adder/ which uses homomorfic encription, but
the project seems abandoned so he is  evaluating Helios:
http://heliosvoting.org/ .

Likely, the application will be built from scracth because terms, legal
conditions, visibility, etc of the system will be specific for spanish
legislation.

The programming languaje, plataform, etc is still undefined.

Hope you have enougth information, there is not much more for now.
