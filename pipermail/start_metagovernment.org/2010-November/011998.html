I would just like to let the list know about this ted talk:

http://www.ted.com/talks/david_bismark_e_voting_without_fraud.html

I think it is a really interesting work.

Some time ago I wrote that when people vote writing all the candidates
in an order, they are essentially giving away their anonimity, in the
sense that someone could force them to vote in a certain way, and
check if they do. That problem still holds. But it solves a lot of
other problems we now have, like the fact that you are normally not
able to verify if your vote has been counted.
