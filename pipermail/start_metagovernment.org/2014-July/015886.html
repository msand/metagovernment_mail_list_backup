On Wed, Jul 23, 2014 at 3:31 PM, Ned Conner <npconner at earthlink.net> wrote:

> I regret that I am still too busy to come out and play, so this will have
> to be just a quick note with no followup:  as a "necessary" precursor,
> could we list and ranked-choice vote on all of the ways that we might
> potentially go about bringing the Home Page project to fruition? My
> preference would be that we continue in Vilfredo. Scott's preference is
> that we up/down vote on one text at a time on Loomio until one wins. Mark's
> preference is that we gather the texts that we have and then ranked-choice
> vote on them. Jacopo's preference is that we use Airesis, and so on.
>
> I for one would find the list and the outcome of that ranked-choice vote
> (on how to decide) of more interest that the vote on home page texts. (We
> demonstrate our competence as a governance research/discussion group
> through HOW we decide, not through WHAT we decide, as far as the Home Page
> text.)
>
>
> On 7/23/2014 2:51 PM, Mark Frischmuth wrote:
>
>>
>> I'm writing to see if we can come to consensus about how to move forward
>> with new text for the homepage.  It seems there have been a number of
>> reasonable alternatives put forward, some on Vilfredo, some on Loomio,
>> perhaps others that I'm not be aware of.  But I don't think there's a
>> shared understanding about how we'll move from a set of alternatives to one
>> formulation on the homepage.
>>
>>
>> A process that I don’t think works is voting alternatives up or down one
>> at a time (as has been attempted on Loomio). This would be similar to
>> asking a child “Do you want cake?” and “Do you want ice cream?”. The more
>> appropriate question is “Do you want cake _OR_ ice cream?”. The child
>> doesn’t get two desserts, and we can’t have more than one description on
>> the homepage.
>>
>>
>>
>> So, we’ll need to decide what how to use to choose between a set of
>> alternatives. I personally like the idea of ranked choice voting, and
>> suggested the possibility of using http://www.opavote.org. Digging into
>> it, it appears suitable for our purpose, except that it’s designed for
>> candidates rather than documents. To use it we might post links to the text
>> of all of the alternatives on the homepage, giving each a short,
>> descriptive name, then below that linking to the voting tool, which would
>> use the same descriptive names as “candidates” in the election. The
>> software allows for the election to last 7 days without charge.
>>
>>
>> If we decide to pursue this approach, I’d suggest we allow 7 days to
>> gather all current formulations from Vilfredo, Loomio, etc. and list them
>> on the homepage. Then allow 7 days for the election itself. If we agree to
>> this approach, we’d have new text for the homepage within two weeks of our
>> start date.
>>
>>
>> What do folks think?
>>
>>
