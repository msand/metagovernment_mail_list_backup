Proposal:

I propose to turn the "home page" of metagovernment into a LIST of links
with "metagovernment homepage proposals", which is ORDERED by the number of
supporters.

Each supporter needs a "Profile Page" (either a profile in the
metagovernment wiki or his personal homepage, where people can read and
confirm, that this person supports a certain proposal for "metagovernment
homepage content"

This way, we can start IMMEDIATELY with putting proposals on the main page
(without losing time while we try to build consensus), but we can still
improve or discuss this.
