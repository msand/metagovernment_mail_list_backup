Perhaps it would make since to conduct the vote on which decision tool to
use on a platform that will not be a choice on the ballot.  For example, I
see this platform (http://civs.cs.cornell.edu/) from Cornell University
that doesn't explicitly state anywhere that it's open source (though it
does say it's written in Perl).  We could list the decision tools we're
considering here, conduct the vote, then use whichever tool wins (using
Condorcet or one of it's variants) to select the homepage text.  I think
excluding the tool being used in this preliminary election from
consideration for the homepage-text decision makes sense, to avoid biasing
decisions with a 'homefield advantage'.
