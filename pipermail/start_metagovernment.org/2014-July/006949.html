About 6 months old, CiteEvidence is "Semantic Wiki of the Month".
http://www.citeevidence.org/wiki/index.php/Main_Page

   Crowd intelligence with the power to empower.  The first browser-
   integrated evidence citation platform for the general public.

   CiteEvidence.org is a fack-check (sic) database in the form of a
   semantic wiki which is used primarily to support the Fact Checker
   browser extension. The wiki collects fact checking results of news
   stories (among other things) in the form of evidential statements
   and the Fact checker downloads and displays this evidence whenever
   and wherever the story is encountered on the Internet. The Fact
   Checker references its own "micro database" of URL signatures so
   there is no need to send browsing history to a remote server to
   check for evidence. In this way, the Fact Checker is autonomous and
   only contacts the CiteEvidence server when it knows there is
   evidence to download for a particular document (the Auto-connect
   function can be turned off). The MicroDB can hold literally
   millions of URL signatures in a compressed format and with the
   power of computers today, this means that clients can take-on more
   processing tasks to maintain anonymity while enjoying the fruits of
   crowd intelligence maintained using an open and all-inclusive wiki
   model. See Fact Checker Basic Operation for more on the ins and
   outs of the Fact Checker.
