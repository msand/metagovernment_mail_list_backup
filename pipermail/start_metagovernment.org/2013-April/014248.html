I recommend you to use GPL licensing
And Creative Commons by-sa 3.0 for Documentation, Wiki, Webpage:
https://creativecommons.org/licenses/by-sa/3.0/deed.en_US

This Creative Commons license is the equivalent of the GPL, but for
license projects that aren't software (this will help you understand
how it works this kind of license)

If is a (integral) program (executed in your computer), GPLv3
If is a library, API, LGPLv3
If is a web application, AGPLv3

Also, follow this links:

About GPL
https://www.gnu.org/licenses/

FAQ
https://www.gnu.org/licenses/gpl-faq.html

list of opensource licenses
this will help you in the reutilization of code, compatibility,
incompabitility, ...
https://www.gnu.org/licenses/license-list.html
