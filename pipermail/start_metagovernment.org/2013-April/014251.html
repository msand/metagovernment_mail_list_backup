I'd like to help.  I've long wanted to see useful comparisons, as I
mention here to Mark Frischmuth:
http://mail.zelea.com/list/votorola/2013-April/001687.html

One question: Could we install the Semantic MediaWiki extension?
http://semantic-mediawiki.org/wiki/Semantic_MediaWiki

SMW would free the authors to design their own comparative tables.  So
one author might design a table that compares all projects (rows) on
properties P, Q, R (columns); while another compares a *subset* of the
same projects on a different set of columns R, S, T.  Note the overlap
in both the rows and columns.  The overlapping cells are automatically
kept in sync by SMW.

It's running in the pollwiki: http://zelea.com/w/Special:Version
But the pollwiki isn't meant for project comparisons, and we have no
plans in that line.  I agree the Metagov wiki is a better place.
