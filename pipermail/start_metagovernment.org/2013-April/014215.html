I have been asked to give a TEDx talk, here in Italy (only 6 minutes or so
:-/ ). I plan to speak about a system I have only dreamed off. :-). It will
be Metagovernment related, but not strictly Metagovernment. I think it will
be great if there are more and more of MG talks on TED(x). Let's just try
not to say exactly the same things. In fact if someone is preparing the
TEDx for Germany, and they accept my abstract for the Italian one, we
should make sure to say different thing, but also refer to each other, so
we start to cast a net, giving the idea that we are all working toward the
same objective :-).
