ZApproved strikes me as being more of a tool for keeping track of
decisions.  I've only read a brief write up on it but it appears to be
something that, out of the box, could be useful to this particular effort.
Might be a way to leverage the work that someone else has done.

It seems like an opportunity to shift our objective from writing software to
actually implement some pilot OpenGov experiments.
