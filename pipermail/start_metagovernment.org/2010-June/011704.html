Hi Kris,

(Thomas is on holiday, so maybe I can help.)  You can easily add
Chennai to the wiki.

 1. Choose a short name (the shorter the better).  Example:

      http://u.zelea.com/w/Che

 2. Create that page, copying the contents from another city:

      http://u.zelea.com/w/DD
      http://u.zelea.com/w/Tor

 3. Then take it from there.  You'll be administering all of the pages
    with names beginning Che/.  You can pretty much do as you please
    in those pages, like it was your own wiki, provided you're doing
    e-democracy (or whatever you choose to call it).

Let us know if we can help with anything,
