When you unzip our package there are two readme files with the 
details of the free engines to download from other web sites and the 
details of invocation.

For those who like drag-and-drop interfaces, once the engines are 
installed all you need do from the Windows file manager interface is 
drag your XML file onto one of the four batch files and you get one 
of four results:

   HTML with the "other information"
   HTML without the "other information"
   PDF single-sided
   PDF double-sided

The presentations are very simple, but the effectively expose the 
content.  The choices made in presentation will be acceptable to many 
as only for review purposes.  That's the nice thing about these 
technologies is that the same XML could be processed by a different 
suite of stylesheets for a very different presentation of the same content.

I hope this helps.  Feedback is very much appreciated.
