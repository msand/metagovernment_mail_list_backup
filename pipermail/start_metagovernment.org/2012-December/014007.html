Thanks Thomas!  Here's a live version.  Please try it folks:
http://metagovernment.org/wiki/Main_Page

   You may need to refresh if it looks funny.  No support for IE yet.
   Only tested on Firefox and Chrome.

It should look the same as the banner mockup:
http://metagovernment.tuxfamily.org/plug/introMock.html
But make allowance for browser settings such as font size.  And
probably only Firefox shows the hover-circles on the switch dial.
Others will show squares.

New visitors should see the banner by default.  Others may need to
hover the mouse over the top left corner to expose the toolset switch.
Turn it to "Introduction".  (The location is obscure as Ed points out,
so I coded the new switch to visibly collapse into the corner when
it's closed.)

What do you think?  Is it okay for starters?  Please report any bugs
or suggestions.
