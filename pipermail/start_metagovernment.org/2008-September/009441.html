This project is progressing nicely, but we could certainly use some  
more help. Following are some things you can do to help it grow.

1. If you are a programmer, please find some time to help out either  
branch of Metascore.

2. If you are not a programmer or don't have time, please contact a  
programmer you know and ask them to consider helping.

3. Post a message about this project on a bulletin board, usenet  
group, list server, or even in the comments section of a blog. More  
links to this project not only raises notoriety, it also increases our  
rank in search engines.

4. Contribute something to the wiki. If you can't think of anything  
off-hand, look through the early discussions in the list server  
archives. There is a lot there that is not yet in the wiki.

5. Make any edit to the wiki, even very minor ones. The more edits  
that show in our Recent Changes list, the more that Wikipedians and  
other sophisticated wiki users respect us when they check out the  
Recent History.

6. Write to someone. Apart from friends and relatives, also search the  
web for technologists or philosophers who might be interested and  
point them to our home page.

7. Reply to this message with further suggestions.
