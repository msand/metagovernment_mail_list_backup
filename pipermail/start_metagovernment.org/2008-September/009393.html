I think the goal of absolute 100% truth is a distraction. If you have
99% accuracy, then that would be wonderful - but if you had something
with only 51% accuracy for truthiness then that'd be better than what
we have now. Plus, a system which is guaranteed to be correct at least
51% of the time, will trend (forever) towards 100% truthiness.
