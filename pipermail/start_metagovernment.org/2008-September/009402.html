Artificial Life measures 'truth' via 'fitness' -- that is, the
'fittest' entites are those which survive in an environment long
enough to pass on more positive than negative traits through genetic
inheritance. The environment doesn't care one way or another, but it
helps shape the entities over time to 'reward' those who work best in
that environment.  Entities in this case could be anything from
simulated wolves and sheep, to stock market predictor agents, etc.

So while the "truthiness" environment doesn't care if we find it or
not, the fact that we can discern some things to be more truthy than
others leads me to believe that we can use that principle of emergence
to our advantage by creating a system that rewards (even if it is only
very slightly at first) truthiness over untruthiness.

Of course, if you get the reward balance the wrong way, you end up
with Fox News, but that's another story - caring about truthiness has
to be a factor.
