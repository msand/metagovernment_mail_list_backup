My two cents here is again drawing a parallel with DemocracyLab's data
structure.  We've created two different types of Comments a user can add to
a Concept.  Users can leave a Comment categorized as Evidence, or a Comment
categorized as Opinion.  This addresses the objective vs. subjective
distinction being discussed below.  All Evidence is expected to be specific
and sourced.  Opinions can be anything a user wishes to write.  

I don't know if this is useful, but I thought it worth sharing.
