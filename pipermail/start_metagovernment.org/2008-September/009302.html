> His problem was with our use of the term "open source governance."

I laughed when I read that. Really if you know anything about Stallman
that is exactly what you would expect. The term Open Source is mainly
for management and marketing people because it seems they can't wrap
there heads around the concept and ideology of free software. I'm
perfectly fine with them using that term because if we do this project
right, we won't have to deal with managers anymore =) Oh happy days.

It's also worth using a different term, because if we go to an
organization and tell them we're about "Open Source Governance" we
might be trying to explain the system in terms of another system that
they don't understand. Then again open source if fairly widespread
these days.

online voting, so I'm not fond of that term.

Governance should be a part of the term I think.
