Perhaps there is an opportunity here when combining the themes of the  
two recent threads on this list. In order for Metagov to organize in  
any way (be it with another group or on our own), we will need to have  
a charter which defines how we operate. Thomas, Conseo, and I started  
with the Draft Rules:
http://metagovernment.org/wiki/User:ThomasvonderElbe/draft_rules
http://zelea.com:8080/v/w/Votespace?p=G!p!m00&u=ThomasvonderElbe%20GmxDe

But we would have to extend this further to cover how we handle money.  
There needs to be some responsibility mechanism for other  
organizations to have any interest in giving us any money.

Michael, does that make sense?

Others, would you object to trying to use Outcast as a basis for the  
governance mechanism of Metagov? The benefit is that Outcast allows  
people to participate in other systems, so by settling on it, we are  
not excluding others.
