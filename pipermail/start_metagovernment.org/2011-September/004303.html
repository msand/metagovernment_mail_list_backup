Crabgrass is a software libre web application designed for social
networking, group collaboration and network organizing.

Features:

    * Decision making tools: approval voting and ranked voting.
    * Personal Messages.
    * Discussions: public or private forums.
    * Wikis: revision history, locking, automatically view changes
since your last visit.
    * File management: upload images, audio, and documents.
    * Task lists.
    * Privacy: every user and group will have control over what
information they choose to reveal about themselves.
    * Security: the first install of crabgrass, hosted at
we.riseup.net, is configured to only allow https connections and all
data is stored in encrypted format.


http://crabgrass.riseuplabs.org
