On Aug 14, 2008, at 1:39 PM, Pietro Speroni di Fenizio wrote:

> do user and up voting on each other tag?
> The way delicious works is that tagging is in itself a way of  
> voting. You put ALL the tags that you consider valid, and then the  
> tag cloud is the emerging measure. Even if someone writes a wrong  
> tag, it makes little weight when compared with all the pther tags  
> everybody else uses.

Yes, I think he is saying that every tag should have a score, and the  
more I think about it, the more sense it makes. (I hope I'm seeing it  
right...) If scores are nuanced, say from 1-10, then people can do  
more than just throw a bunch of tags at something: they can say this  
tag applies a lot and that tag applies a little.

I don't know if anyone has thought of that before: combining tags and  
scores. The mind boggles at the possibilities this opens, even though  
it is a rather simple twist.

My main question is technical: do we *require* a score for every tag,  
or make a default (but community-specified) score for tags which are  
not given a score by the user?
