It would make sense for the very first test case to be ourselves: to  
govern the Metagovernment project with Metascore. I imagine we will  
have to do it as a provisional government: when things don't work out,  
the list server or individuals may supersede the Metascore instance.  
Once we get to a point where we can reliably always use Metascore,  
then it would be wonderful to have something like Balseiro as a larger  
test sample.

>> advantage contrasted to predict it. Test fields will give us the
>> feedback that we need to make discoveries.
>>
